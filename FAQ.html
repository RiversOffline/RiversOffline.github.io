This website utilizes your browsers cache to store the contents of this site - So whenever rivers.run is in cache, you can visit it. If for any reason your cache is cleared, then you will be unable to visit rivers.run without internet until the cache is refilled on the next visit.

The URL to a normal search is https://rivers.run#YOUR_SEARCH_QUERY_HERE
If you want to test the URL, make sure you put it in a new tab.

If are encountering issues, your browser may be too old. Here are the browsers that should work:
Chrome (Version 40ish+ should work, but only the latest version tested)
Firefox - Full Support can be obtained down to Firefox 60 ESR. river-overview popups break below version 60
Microsoft Edge (Chakra) - Full Support believed for latest version
Opera, Microsoft Edge (Blink), Chromium - Based on the same engines as Chrome, so they should work on the latest version, and down to Chromium 40ish+
Opera Mini - Breaks in extreme mode. Works mostly otherwise (navigation menu dissapears though)
Samsung Internet - Based on WebKit just like Safari, so it should work
Any Browser on iOS 9.3+ (iOS 11.3+ needed for offline)
