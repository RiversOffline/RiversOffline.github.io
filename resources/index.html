Resources

Resources to help with development of Rivers.run


writeupmaker.html
Used to create the River Writeups in the format needed for dataparse.html

dataparse.html
dataparse.html is used to generate RiverArray.js, the file that contains the river data for rivers.run
It fetches the latest river data from Google Docs using the Google Drive API. The data is then converted into the format used by rivers.run
I then download the file that it generates and upload it to github.


This domain also stores some image files to avoid cluttering up the Rivers.run Github repository.
